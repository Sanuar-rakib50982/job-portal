# Group Git Workflow

Each member must work in a separate feature branch.

Branches:
- feature/admin
- feature/job-seeker
- feature/employer
- feature/recruiter

Rules:
- Do not push directly to main.
- Commit regularly.
- Push your own branch.
- Create Pull Request to main.
- Group leader reviews and merges.
- Do not edit another member's role files without permission.
