# Database Setup

1. Start XAMPP Apache and MySQL.
2. Open phpMyAdmin.
3. Create a database named `job_portal`.
4. Import `database/schema.sql`.
5. Confirm required tables exist:
   - users
   - jobs
   - categories
   - applications
   - saved_jobs
   - complaints
   - messages
   - job_alerts
   - recruiter_outreach
   - platform_settings
   - announcements
