# Admin Testing Checklist

- Admin login works
- Dashboard summary loads correctly
- Recent users show correctly
- User search and filters work
- Employer approval works
- Recruiter approval works
- User suspension and reactivation work
- Category create, edit, and delete work
- Category delete is blocked when active jobs exist
- Jobs page loads correctly
- Featured job toggle works through AJAX
- Complaint resolve and reopen work
- Platform policies update correctly
- Announcements can be created and deleted
- Analytics page loads correctly
- CSV report export works
- Non-admin users cannot access admin pages
